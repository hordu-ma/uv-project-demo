## 🤖 Copilot 代理功能

本项目实现了 **"delegate to coding agent"** 概念，允许主 Copilot 系统将复杂编程任务委托给专门的编码代理：

### 核心组件

- **`Copilot`**: 主要协调器，负责理解用户意图并委托任务
- **`CodingAgent`**: 抽象基类，定义编码代理接口
- **专门化代理**:
  - `CodeGenerationAgent`: 专门负责代码生成（函数、类、代码片段）
  - `UnitTestGenerationAgent`: 专门负责单元测试生成
  - `CodeAnalysisAgent`: 专门负责代码分析和质量检查

### 使用示例

```python
from uv_project_demo import (
    Copilot, TaskRequest, 
    CodeGenerationAgent, UnitTestGenerationAgent, CodeAnalysisAgent
)

# 初始化 Copilot 并注册代理
copilot = Copilot()
copilot.register_agent(CodeGenerationAgent())
copilot.register_agent(UnitTestGenerationAgent())
copilot.register_agent(CodeAnalysisAgent())

# 创建任务请求
task = TaskRequest(
    task_type="function",
    description="Create a function to calculate factorial",
    context={"name": "factorial", "parameters": ["n: int"], "return_type": "int"}
)

# 委托任务
result = copilot.delegate_task(task)
if result.success:
    print(f"Generated by: {result.metadata['handled_by']}")
    print(result.result)
```

### 运行演示


### 核心优势

- **分工明确**: 每个代理专注于特定类型的编程任务，提升效率和准确性
- **提升效率**: 任务自动路由到最合适的代理
- **可扩展性**: 容易添加新的代理类型
- **错误处理**: 优雅处理不支持的任务类型
